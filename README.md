**Pokémon Megamax** is a fan-made Pokémon RPG built with **RPG Maker XP** and **Pokémon Essentials**. Inspired by the beloved (but now-defunct) Roblox game *Pokémon Brick Bronze*, this project reimagines that adventure with a brand-new region, a mature dark storyline, Gen IV/V-styled graphics, and deeply expanded gameplay systems. This repository houses the **official promotional website** for the game — a polished multi-page HTML/CSS site showcasing story details, features, a screenshot gallery, download information, and developer credits.

## ✨ Key Features

- 🗺️ **Brand-New Region** — Explore a fully custom region with original maps and environments crafted from scratch.
- 📖 **Story-Driven Gameplay** — A rich narrative with in-depth cutscenes, a companion character (Jake), and unexpected twists.
- 🎨 **Gen IV / V Graphics** — Custom UI, overworld sprites, and tilesets modelled after the 4th and 5th Generation art style.
- 💾 **Multiple Save Files** — Save your journey across multiple slots.
- 🌑 **Dark Storyline** — Mature themes including violence and on-screen consequences. Viewer discretion is advised.
- 🧬 **Mutant Pokémon** — Encounter unique mutant variants like the fearsome Mutant Trevenant.
- 🔧 **Custom UI** — Redesigned menus, battle interfaces, and HUD elements.
- ⚡ **Rebuilt from Scratch** — Beta 1.3 is a complete rewrite for improved performance and stability.

## 📜 Story

You play as a young Pokémon trainer whose journey takes a dramatic turn when the ship you're travelling on is wrecked in a catastrophic accident. Saved from drowning by the Legendary Pokémon **Necrozma** — an eccentric yet kind being — your adventure in a brand-new region begins in the most unexpected way.

Along the way you'll be joined at times by your childhood friend **Jake**, face moral dilemmas, witness unexpected events, and ultimately clash with **Team Saber** — an anarchistic criminal organization that weaponises Pokémon for their sinister schemes.

> ⚔️ **Content Warning:** Some story sequences contain violence, bloodshed, and on-screen deaths. Viewer discretion is advised.

### Key Story Characters

| Character | Role | Description |
|---|---|---|
| **Protagonist** | Player Character | A young trainer shipwrecked at sea, beginning a fateful new adventure. |
| **Jake** | Rival / Ally | Your childhood friend who accompanies and occasionally challenges you. |
| **Necrozma** | Legendary / Ally | Rescues the protagonist; eccentric among Legendaries but genuinely kind. |
| **Team Saber** | Antagonist Faction | A ruthless anarchistic criminal group exploiting Pokémon for power. |

## 🤝 Contributing

This repository contains the **promotional website** source code. Contributions to the site are welcome!

```bash
# 1. Fork the repository
git clone https://github.com/Xen-Vortexu/Pokemon_Megamax-updated.git

# 2. Navigate to the project
cd Pokemon_Megamax-updated

# 3. Open index.html in your browser — no build step required!
open index.html
```

**Guidelines:**
- Keep pages consistent with the existing HTML/CSS style conventions
- Test changes across major browsers (Chrome, Firefox, Edge)
- Open an issue before making large structural changes
- For game content contributions, join the Discord `#👍-recruitment` channel
